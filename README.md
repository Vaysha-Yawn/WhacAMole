# Whac-A-Mole
It is a Whac-A-Mole game.
Created for a test task in the company ABC Team.

#### Rules of the game
Hit the mole with the maximum number of hits in 30 seconds.

## Technology

### Programming language
- Kotlin

### Libraries

- Jetpack Compose
- Coroutine
- Lifecycle
- SharedPreferences
